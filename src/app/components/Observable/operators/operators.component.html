Operators

Operators in rxjs are functions that takes an observable as input and transforms it into a new observable and returns it. We use operators to manipulate the observable data stream.

Operators used here from the rxjs library

  1. Map - Applies a given project function to each value emitted by the source Observavle, and emits the resulting values as an Observable.
  2. Filter - Filter items emitted by the source Observable by only emittin those that satisfy a specified condition or predicate
