Pure Pipes

Go to the following Pipe to see how Pure Pipes work. Here are the instructions to view the code...

  1. In the directory... go to the following path: CustomPipes/percentagesPipe.ts
  2. You will see the value for pure here under
    @Pipe((
    name: 'percentage',
    pure: false
    ));

    * If you don't explicity specify that this is a pure pipe, it will render as pure unless you set the value to false

What is a Pure Pipe?

Pure pipes are those pipes which gets executed when Angular detects a pure change on the input value.

Example:
<div> { { std.gender | lowercase } } </div>

Pure Pipes are only needed if a Pure Change has occured

A "Pure Change" is a change of a primitive input value (String, Boolean, Number, etc.)

A "Impure Change" are pipes that get executed when Angular detects a pure change on the input value

Go to Custom Pipes page to see example here: Custom Pipes

Also, the example of the pipe being used can be found in the percentage pipe. Look for it in the following directory:
/CustomPipes/percentage-pipe.ts

It should look like this:

import { Pipe, PipeTransform | from '&angular/core';
@Pipe({
name: 'percentage',
|)
export class PercentagePipe implements PipeTransform {
transform(value: any) {
return value * 100;
}
}
