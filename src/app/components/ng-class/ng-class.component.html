Ng Class

Uses a class name referenced in the template css file and is based on a condition in the HTML element in which that style is being applied.


Change the color of this text using ngClass if the value is true!

Don't change the color of this text using ngClass if the value is false!

HTML FILE:
<p [(ngModel)]="value" [ngClass]="{ changeBg: value === true }"> Change the color of this text using ngClass if the value is true! </p>

TYPESCRIPT FILE:
public value: boolean = true;
