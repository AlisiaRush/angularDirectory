Reactive Forms

  1. Reactive forms are forms where we define the structure of the form in the component class. i.e., we create the form model with Form Groups, Form Controls, and Form Arrays.
  2. We also define the validation rules in the component class.
  3. Then, we bind it to the HTML form in the template.
  4. This is different from the template-driven forms, where we define the logic and controls in the HTML template.
